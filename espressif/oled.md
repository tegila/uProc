###OLED 128x64

|:----:|:----------:|:----:|
| SCLK |   GPIO14   |  D0  |
| MOSI |   GPIO13   |  D1  |
| MISO | (not used) |   -  |
|  !SS |   GPIO15   |  CS  |

|   -  |   GPIO16   |  RST |
